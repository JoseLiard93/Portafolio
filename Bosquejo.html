  1. Descripcion

    2. Discusión de conceptos de diseño y desarrollo de páginas Web. Aplicación de los
    lenguajes HTML y JavaScript conjuntamente con CSS para la creación de páginas
    Web. Creación y adaptación de elementos de multimedios para ser utilizados en
    páginas interactivas. Requiere 45 horas de conferencia-laboratorio. 3 créditos

  1. Objetivos

    2. Se espera que al finalizar el curso el estudiante pueda:

      1. Discutir conceptos de diseño y desarrollo de páginas Web.
      2. Aplicar los lenguajes HTML y JavaScript conjuntamente con CSS para la
        creación de páginas Web.
      3. Crear y adaptar elementos multimedios para ser utilizadas en páginas
        interactivas.
      4. Entender las implicaciones éticas, sociales y legales inherentes a la disciplina.

  1. Contenido Tematico

    1. Conocer los editores del lenguaje HTML
      1. Historia de los editores de HTML
      2. Introduccion a HTML
        1. Listas numeradas, no numeradas y de definicion
        2. Enlaces internos y externos
        3. Incorporacion de tablas, imagenes y videos
        4. Diseño de formulario e incororacion
    1. Creacion de Paginas interactivas
      1. Creacion de paginas interactivas utilizando un editor de HTML
      2. Publicacion de paginas interactivas
    1. Introduccion a CSS
      1. Uso de CSS
      2. Reglas de CSS
      3. Incorporacion de CSS a una pagina interactiva
    1. Introduccion a JavaScript
      1. Definicion
      2. Incorporacion de un javascipt a paginas interactivas
      3. Integracion de elementos visuales que utilizan animacion vectoriales
